# CS1 Benchmark Specification

## Benchmark Name

CS1 Code Judgment Benchmark (Pilot Version)

## Purpose

The benchmark is designed to evaluate the reliability and consistency of AI-generated Python programs for introductory programming (CS1) problems.

## Target Learners

Novice programmers enrolled in introductory programming courses.

## Programming Language

Python 3

## Topics Covered

- Variables
- Input and Output
- Operators
- Selection
- Loops
- Functions
- Lists
- Strings
- Dictionaries
- Recursion

## Difficulty Levels

Easy

Medium

## Number of Problems

Pilot Benchmark: 10

Full Benchmark (Planned): 100–150

## Evaluation Method

Each AI-generated program will be evaluated using hidden test cases.

## Evaluation Metrics

- Syntax correctness
- Runtime correctness
- Hidden test case pass rate
- Functional correctness
- Error category
- Code consistency
